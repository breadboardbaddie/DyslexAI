# DyslexAI — Companion Web App

Live: https://dyslexai-five.vercel.app

Landing page for the DyslexAI Chrome extension. Built with Next.js + Tailwind, deployed on Vercel.

## Development

```bash
npm install
npm run dev
```

## Deploy

```bash
npx vercel --prod
```

See the [main README](../README.md) for full project documentation.
